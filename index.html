Introduction¶

yaps2 is a loose collection of data processing pipelines meant the CCDG project. All of these pipelines are currently designed to be run internally at the McDonnell Genome Institute (MGI).

There are currently 4 available pipelines:

  1. postvqsr

    This is a pipeline to further post-process VCF files that have been generated from GATK’s Variant Quality Score Recalibration (VQSR) pipeline.

  2. postvqsr38

    Similiar to the postvqsr pipeline, but meant to work on VCF files based on Human Reference Build version 38.

  3. Mendelian Inheritance Errors (mie)

    A niche data processing pipeline to examine the relationship between Mendelian Inheritance Errors to the VQSR score for a selected set of genotypes.

  4. Principal Component Analysis (pca)

    A niche data processing pipeline to perform principal component analysis on a selected set of genotypes.

  5. Build 38 realignment

    A data pipeline to realign BAM / CRAM files on Build 38 via speedseq.
